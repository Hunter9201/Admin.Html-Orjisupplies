Set GitHub Token

IMPORTANT: You need a GitHub Personal Access Token with repo permission.

How to create token:

  1. Go to GitHub Settings → Tokens
  2. Click "Generate new token" → "Generate new token (classic)"
  3. Give it a name like "Orji Supplies Admin"
  4. Select repo permission (full control of private repositories)
  5. Click "Generate token"
  6. Copy the token and paste it here

Note: Token is stored only in your browser.
